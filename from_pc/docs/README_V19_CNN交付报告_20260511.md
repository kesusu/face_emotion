### 5.2 Expert集成预期

当前Expert v5.1 = 64.9%:
- SVM7路由 + V15 CNN(强类) + SVM3+RF(弱类)

替换为V19f后:

3. **恐惧类弱(6.1%)** — 样本太少(332张), 建议仍由路由分配给级联处理

## 六、PC端训练经验（给板端参考）

### 6.1 7次训练的关键教训

| 版本 | Acc | 问题 | 原因 |
|------|-----|------|------|
| V19a | 8.68% | 严重欠拟合 | dropout=0.65, WD=5e-3, LS=0.15 太强 |
| V19c | 20.0% | 厌恶偏83% | FocalLoss alpha + WeightedSampler 双重平衡 |
| V19d | 41.4% | 中性偏81% | 单Sampler 1/count权重太极端 |
| V19e | 47.6% | 恐惧0%, 厌恶0.6% | 无类别平衡, 小类崩 |
| **V19f** | **47.9%** | **当前最佳** | **sqrt权重 + FocalLoss(gamma only)** |

### 6.2 类别平衡黄金法则

```
❌ 双重平衡(FocalLoss alpha + Sampler) = 矫枉过正
❌ 极端权重(1/count) = 多数类被过度压制
❌ 无平衡 = 小样本类(恐惧/厌恶)完全崩溃
✅ 温和平衡(sqrt权重 + FocalLoss仅gamma) = 各类均衡
```

### 6.3 CNN天花板≈48%

热力图类间cos_sim差仅0.005, CNN卷积核找不到边缘/纹理区分信号。
48%可能是CNN在板端SDK热力图上的天花板, 继续调参边际收益极低(+0-1%)。

### 6.4 突破方向

- **3类专用CNN(厌恶/悲伤/中性)**: 预期悲伤+15-20pp, 任务书已有方案
- **特征工程**: 热力图统计量(mean/std/skew)可能比像素更有效
- **梯度提升树**: XGBoost/LightGBM可能比RF再+1-2%
