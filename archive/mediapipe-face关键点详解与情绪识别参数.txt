一、模型输出格式
────────────────

输入: 一张包含人脸的图像 (numpy数组 / 图片路径)
输出: 468个关键点，每个点包含:
  - x: 水平坐标 (归一化 0~1，相对图像宽度)
  - y: 垂直坐标 (归一化 0~1，相对图像高度)
  - z: 深度坐标 (相对脸部中心，负值=靠近摄像头)

坐标系:
  - x=0 图像左边, x=1 图像右边
  - y=0 图像顶部, y=1 图像底部
  - z 负值表示该点比脸部中心更靠近摄像头

转像素坐标:
  pixel_x = landmark.x * image_width
  pixel_y = landmark.y * image_height

★★★  四、情绪识别特征计算公式（纯算法，无需LLM）  ★★★
================================================================================

以下公式中的坐标均为归一化坐标(0~1)，需要做面部归一化消除距离影响。

────────────────────────────────────
特征1: 眉毛高度 (Eyebrow Height)
────────────────────────────────────
  左眉高度 = 点105.y - 点33.y   (眉中点相对于左眼外角)
  右眉高度 = 点334.y - 点263.y  (眉中点相对于右眼外角)
  眉毛高度 = (左眉高度 + 右眉高度) / 2

  → 数值越大(越负) = 眉毛越高 = 惊讶/恐惧
  → 数值越小(越正) = 眉毛越低 = 愤怒

────────────────────────────────────
特征2: 眉间距离 (Inter-brow Distance)
────────────────────────────────────
  眉间距离 = |点70.x - 点300.x|

  → 数值越小 = 皱眉 = 愤怒
  → 数值越大 = 眉毛舒展 = 放松/开心

────────────────────────────────────
特征3: 眼睛开合度 (Eye Openness) — ★最重要
────────────────────────────────────
  左眼开合度 = |点159.y - 点145.y| / |点33.x - 点133.x|
  右眼开合度 = |点386.y - 点374.y| / |点263.x - 点362.x|
  眼睛开合度 = (左眼开合度 + 右眼开合度) / 2

  → 数值大 = 眼睛大睁 = 惊讶/恐惧
  → 数值中等 = 正常
  → 数值偏小但嘴角上扬 = 笑眼(开心)
  → 数值小 + 上眼睑下垂 = 悲伤/困倦

────────────────────────────────────
特征4: 嘴角高度 (Mouth Corner Height) — ★最重要
────────────────────────────────────
  参考线 = (点0.y + 点17.y) / 2   (唇线中点)
  左嘴角偏移 = 点61.y - 参考线    (正值=嘴角下垂, 负值=嘴角上扬)
  右嘴角偏移 = 点291.y - 参考线
  嘴角高度 = (左嘴角偏移 + 右嘴角偏移) / 2

  → 数值为负 = 嘴角上扬 = 开心
  → 数值为正 = 嘴角下垂 = 悲伤
  → 数值接近0 = 嘴角平直 = 中性/愤怒

────────────────────────────────────
特征5: 嘴巴开合度 (Mouth Openness)
────────────────────────────────────
  嘴巴开合度 = |点13.y - 点14.y| / |点61.x - 点291.x|

  → 数值大 = 嘴巴大张 = 惊讶
  → 数值中等 = 嘴巴微张 = 恐惧/说话
  → 数值小 = 嘴巴闭合 = 中性/愤怒/悲伤
  → 数值小+嘴角上扬 = 微笑

────────────────────────────────────
特征6: 鼻翼扩张 (Nose Wing Expansion)
────────────────────────────────────
  鼻翼宽度 = |点98.x - 点327.x|
  鼻梁宽度 = |点48.x - 点278.x|  (参考基准)
  鼻翼比 = 鼻翼宽度 / 鼻梁宽度

  → 数值大 = 鼻翼扩张 = 厌恶/愤怒

────────────────────────────────────
特征7: 下巴下垂 (Jaw Drop)
────────────────────────────────────
  下巴下垂度 = (点152.y - 点17.y) / |点10.y - 点152.y|
  (下巴点相对于下唇到额头距离的比例)

  → 数值大 = 下巴下移 = 惊讶(大张嘴)

────────────────────────────────────
特征8: 眉毛内端上抬 (Inner Brow Raise)
────────────────────────────────────
  左眉内端高度 = 点70.y - 点33.y
  右眉内端高度 = 点300.y - 点263.y
  眉内端抬升 = (左眉内端高度 + 右眉内端高度) / 2 - 眉毛高度

  → 眉内端比整体更高 = "八字眉" = 悲伤
  → 眉内端比整体更低 = 皱眉 = 愤怒

七、在另一台设备上开发调试的方法
================================================================================

1. 安装 mediapipe (跨平台，输出格式与板端 .fmodel 完全一致):

   pip install mediapipe opencv-python

2. 示例代码 (Windows/Mac/Linux 通用):

   import cv2
   import mediapipe as mp

   mp_face_mesh = mp.solutions.face_mesh
   face_mesh = mp_face_mesh.FaceMesh(
       max_num_faces=1,
       refine_landmarks=True,   # 包含虹膜点(468-477)
       min_detection_confidence=0.5,
       min_tracking_confidence=0.5
   )

   img = cv2.imread("test_face.jpg")
   rgb = cv2.cvtColor(img, cv2.COLOR_BGR2RGB)
   results = face_mesh.process(rgb)

   if results.multi_face_landmarks:
       landmarks = results.multi_face_landmarks[0].landmark
       # 读取第159号关键点(左眼上眼睑)
       p = landmarks[159]
       print(f"x={p.x:.4f}, y={p.y:.4f}, z={p.z:.4f}")

3. 算法调通后，移植到板端只需把 mediapipe 推理换成:

   result = cv.InferCvKeypointSync(image)
   # result 的关键点坐标含义与 mediapipe 完全一致

   关键点索引号、坐标含义、计算公式全部通用，无需任何修改。


================================================================================
八、补充说明
================================================================================

1. 本模型运行在 GPU 上(文件名含 gpu)，非 DSP，推理速度约 10~20ms/帧
2. 坐标为归一化坐标(0~1)，计算特征前必须做归一化（除以面部尺寸），
   否则远近不同会导致数值差异巨大
3. z 坐标不稳定，建议情绪识别只用 x/y，z 仅做辅助验证
4. 建议用连续3~5帧做平滑处理，单帧判断容易抖动
5. 老年人面部特征（皱纹、皮肤松弛）可能导致部分特征偏移，
   建议实际采集数据后微调阈值
6. 6种基本情绪之外可增加 "中性(neutral)" 作为默认状态
