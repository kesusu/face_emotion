## §1 — 01_AI项目交接文档_给开发板.md

> 项目概况、系统架构、核心文件清单、准确率水平、10大技术突破、关键技术细节、待办事项、经验教训

_（完整内容取自此文件）_

（以下是原始文件内容，包含：项目背景、使用的硬件平台、MediaPipe vs SNPE选择、PyTorch模型结构、实验记录、准确率数据等）

### 1.1 项目概况
- **任务**: 基于人脸468个关键点，实时识别7类基本情绪（惊讶、恐惧、厌恶、开心、悲伤、愤怒、中性）
- **开发板**: qcs6490 (高通QCS6490), ARM64
- **PC端**: RTX 4060 Laptop GPU (8GB), 用于训练
- **数据集**: RAF-DB (Real-world Affective Faces Database)

### 1.2 系统架构
```
板端运行流程:
摄像头 → 人脸检测 → MediaPipe FaceMesh(468点)
    → 14维手工特征(规则/SVM) + 热力图(CNN)
    → Ensemble融合 → 输出7类概率
```

### 1.3 核心文件
| 文件 | 用途 | PC/板 |
|------|------|-------|
| run_face_keypoint.py | 板端主推理程序 | 板 |
| cnn_heatmap_classifier.py | CNN热力图训练 | PC |
| pc_infer_adapter.py | PC端推理适配(MediaPipe) | PC |
| retrain_svm.py | SVM重训脚本 | 板 |

### 1.4 V8~V9关键数据
| 模型 | 准确率 | 说明 |
|------|--------|------|
| V5规则系统 | ~23% | if-else规则基线 |
| V6规则+反吞噬 | ~23%(复合37%) | 弱类从0→17% |
| V8 SVM(Linear,14维) | 54.5% | 12种ML对比 |
| V9 CNN(64×64热力图) | 67.7%(板端实测) | SGD+Nesterov |
| V10 Ensemble | 72.0% | 三路加权融合 |

### 1.5 10大技术突破
1. 三通道语义分解热力图编码（XY+Z+区域权重）
2. 多模态异构融合（规则+SVM+CNN）
3. 端云协同流水线（PC GPU训练→板端部署）
4. 跨引擎兼容（MediaPipe ≈ fiboaisdk, LM 99.7%）
5. 诊断驱动迭代（Cohen's d定位根因）
6. 四层架构分析法（数据→特征→模型→架构）
7. 经典结论优先（SGD > Adam）
8. 反吞噬守卫（避免规则误判）
9. ROI驱动优先级
10. AI方法论驱动AI开发

## §3 — 04_AI_CNN训练报告_PC端.md

> 6轮GPU训练实验记录、V9.5 SGD最佳模型40.0%、各类别准确率

### 3.1 V9.5 SGD最佳结果
| 类别 | 准确率 |
|------|--------|
| 惊讶 | 90% |
| 恐惧 | 91% |
| 厌恶 | 33% |
| 开心 | 27% |
| 悲伤 | 22% |
| 愤怒 | 38% |
| 中性 | 20% |
| **整体** | **40.0%** |

### 3.2 训练历程
| 轮次 | 模型 | ValAcc | 说明 |
|------|------|--------|------|
| V9.1 | DeepCNN AdamW, LS=0.1 | 36.2% | 基线 |
| V9.2 | DeepCNN SGD, LS=0.1 | 37.2% | +1.0pp |
| V9.3 | DeepCNN SGD, LS=0 | 37.5% | +0.3pp |
| V9.4 | DeepCNN SGD+nesterov, LS=0, aug | 37.7% | +0.2pp |
| **V9.5** | **DeepCNN SGD+nesterov, LS=0, aug, 120ep** | **40.0%** | **+2.3pp** |
| V9.6 | ResNet18, Aug | 25.7% | 过拟合，放弃 |

### 3.3 关键结论
- SGD+Nesterov > AdamW: +1.7pp
- Label Smoothing: -10pp（小数据集不适用）
- V9.5受限于64×64分辨率，128×128预期更高

## §5 — _HANDOVER_20260509.md

> 数据清洗后V16(SVM 56.0%)、V17三类专用SVM、ADB推送规范、完整时间线

### 5.1 关键突破
| 模型 | 准确率 | 说明 |
|------|--------|------|
| V15 CNN | 30.6% | 板端SDK重训 |
| V16 RF(PCA50) | **47.2%** | 新发现：ML > CNN |
| V16 SVM7(RBF) | 41.7% | 7类 |
| V17 3类SVM | **54.5%** | 强弱分离 |
| 历史SVM(14维手工特征) | **54.5%** | 历史最强 |

### 5.2 CNN路线被否定
- 热力图类内cos_sim=0.97, 类间=0.965, 差异仅0.01
- CNN(480K参数)严重过拟合(Train 67% vs Val 38%)
- **结论: 热力图信息高度集中，前20成分覆盖99.85%方差**

### 5.3 新路线图
1. CNN路线放弃 → 转RandomForest/SVM
2. 分类设计: "弱方法权重降低→提升整体"
3. Ensemble预期: SVM+RF+规则 ≈ 55-60%
