### 2. 热力图样例（用于 MediaPipe 一致性验证）⭐ 建议

> **目的**: 对比PC端和板端MediaPipe生成的热力图是否一致。如果差异大，说明两端模型输出不同，需统一。

| 来源 (PC) | 目标 (板子) | 说明 |
|-----------|------------|------|
| `heatmap_verify/` 整个目录 | 对应目录 | 每类别2个样本 + 对应的.npy |

**验证方法**:
```
板子上: 用自己的MediaPipe对同样几张jpg生成热力图 → 和PC推送过去的.npy逐像素对比
如果误差 < 0.01 → 一致 ✅
如果误差大 → 两端MediaPipe版本不同，需统一
```
