- 全7类CNN，ValAcc > 55%（当前V15仅30.6%）

## 二、五次训练尝试及教训

### V19a — 严重欠拟合 (8.68%)

| 参数 | 值 | 问题 |
|------|-----|------|
| dropout | 0.65 | 太强 |
| weight_decay | 5e-3 | 太强 |
| label_smoothing | 0.15 | 太强 |
| lr | 5e-4 | 太低 |
| 卷积层Dropout2d | 有(0.15~0.2) | 额外正则化 |
| **结果** | **ValAcc=8.68% @ ep38** | **严重欠拟合！** |

**教训**: 正则化全拉满 = 模型根本学不动。

---

### V19b — 被中断

电脑意外重启，训练中断。

---

### V19c — 厌恶偏严重 (20.0%)

| 参数 | 值 |
|------|-----|
| dropout | 0.6 |
| weight_decay | 1e-4 |
| label_smoothing | 0.1 |
| lr | 1e-3 |
| scheduler | CosineAnnealing(T=50) |
| FocalLoss | alpha(类别权重)+gamma=2.0 |
| WeightedSampler | 1/count权重 |
| **结果** | **ValAcc=20.0% @ ep28** |

混淆矩阵: 模型严重偏向"厌恶"，其他类大量被误判为厌恶。

**根因**: FocalLoss alpha + WeightedSampler = 双重类别平衡，矫枉过正！

---

### V19d — 中性偏严重 (41.4%)

| 参数 | 值 |
|------|-----|
| 同V19c | 但去掉FocalLoss alpha |
| 损失函数 | CE + LabelSmoothing |
| WeightedSampler | 1/count权重(保留) |
| **结果** | **ValAcc=41.4% @ ep12** |

中性81%但悲伤0%，厌恶9.2%。单WeightedSampler仍让中性占主导。

---

### V19e — 纯基线无平衡 (47.6%) ★ 之前最高

| 参数 | 值 |
|------|-----|
| 损失函数 | CE + LabelSmoothing(0.1) |
| 采样 | 普通shuffle, 无WeightedSampler |
| 无FocalLoss, 无MixUp | |
| **结果** | **ValAcc=47.6% @ ep49** |

逐类: 中性88.6%、开心45.5%、惊讶46.3%、愤怒41.1%、悲伤19.4%、**恐惧0%、厌恶0.6%**
问题: 不加类别平衡, 小样本类(恐惧66/厌恶173 val)完全崩

---

### V19f — 温和平衡 (47.9%) ★★ 当前最佳

| 参数 | 值 |
|------|-----|
| 损失函数 | FocalLoss(gamma=2.0, **无alpha**) + LabelSmoothing(0.1) |
| 采样 | WeightedRandomSampler(**根号权重**: 1/sqrt(count)) |
| **结果** | **ValAcc=47.9% @ ep67** |

逐类:
```
惊讶: 50.5%    恐惧:  6.1%    厌恶: 19.7%    开心: 39.6%
悲伤: 16.1%    愤怒: 50.3%    中性: 83.6%
```

**vs V19e对比**: 恐惧+6.1pp, 厌恶+19.1pp, 愤怒+9.2pp — 小类显著恢复
但悲伤(16.1%)和中性(83.6%)不均衡，中性仍占主导。

---

### V19g — 解决中性偏执 (3版本, 2026-05-12)

> **任务书**: 板端反馈V19f在Expert集成后整体暴跌16pp(64.6%→48.3%)
> **根因**: V19f把过半厌恶(53%)和悲伤(57%)判成中性，污染了Expert的强类路由

#### V19g-a — 激进降权 (40.2%) ★★ 交付版

| 参数 | 值 | vs V19f |
|------|-----|---------|
| FocalLoss alpha | 中性=0.5, 恐惧=1.5, 厌恶/愤怒=1.2 | V19f无alpha |
| 中性样本截断 | 3074→1500 | V19f全量3074 |
| label_smoothing | 0.05 | V19f=0.1 |
| dropout | 0.5 | V19f=0.6 |
| **结果** | **ValAcc=40.2% @ ep36** | V19f=47.9% |

混淆矩阵关键: 非中性→中性max=2.2% (V19f=56%!)
```
         惊  恐  厌  开  悲  愤  中
  惊讶 [ 192  12  25  14  61   4   7]  → 中性2.2% (V19f=33%)
  恐惧 [  13  16   4  13  14   5   1]  → 中性1.5% (V19f=33%)
  厌恶 [  28   3  66  20  50   4   2]  → 中性1.2% (V19f=53%)
  开心 [  12  15  16 134  40   5   0]  → 中性0.0% (V19f=23%)
  悲伤 [  58  19  61  80 186  17   7]  → 中性1.6% (V19f=57%)
  愤怒 [  22   9  21  15  31  65   0]  → 中性0.0% (V19f=17%)
  中性 [  69   1  55  23 136   5  11]  → recall=3.7% (V19f=87%)
```

**关键发现**: ValAcc降了7.7pp，但中性偏执从56%降到2.2%。对Expert集成可能更有利。

#### V19g-b — 极端降权 (33.2%) 过度

| 参数 | 值 |
|------|-----|
| FocalLoss alpha | 中性=0.3 |
| label_smoothing | 0.0 (关闭) |
| dropout | 0.5 |
| **结果** | **ValAcc=33.2% @ ep26** |

中性recall=0%! 模型完全不预测中性了。混淆矩阵中列全是0。
**教训**: alpha=0.3太低，中性被彻底压制，失去识别能力。

#### V19g-c — 温和降权 (32.4%) 效果弱

| 参数 | 值 |
|------|-----|
| FocalLoss alpha | 中性=0.7 |
| sampler中性降权 | ×0.5 |
| dropout | 0.6 (同V19f) |
| label_smoothing | 0.1 (同V19f) |
| **结果** | **ValAcc=32.4% @ ep42** |

中性recall=1.0%, 非中性→中性max=0.5%。
**问题**: sampler降权×0.5 + alpha=0.7双重压制虽然温和，但ValAcc太低。

### V19g 三版本对比

| 版本 | ValAcc | 非中性→中性max | 中性recall | alpha[6] | 截断 | LS |
|------|--------|---------------|-----------|----------|------|-----|
| **V19g-a** | **40.2%** | **2.2%** | 3.7% | 0.5 | 1500 | 0.05 |
| V19g-b | 33.2% | 0.0% | 0% | 0.3 | 无 | 0 |
| V19g-c | 32.4% | 0.5% | 1.0% | 0.7 | 无(sampler×0.5) | 0.1 |

**结论**: alpha=0.5 + 截断1500是最佳平衡点。alpha太低(0.3)压死中性，太高(0.7)效果弱。

### V19g-b/c — 温和降权重训 (2版本, 2026-05-12 下午)

> **根因**: V19g-a alpha=0.5+截断1500太激进，中性压死但决策边界也退化

**强类acc不达标**: 恐惧19.7%, 厌恶22.5%, 愤怒46.0% (目标>50%)

## 三、核心发现

### 3.1 CNN在热力图上的天花板 ≈ 48%

5次训练从8.68%到47.9%，核心趋势：
- 去掉过度正则化 → 30→48%
- 类别平衡太强 → 偏类(20%)
- 无类别平衡 → 小类崩(恐惧0%)
- 温和平衡 → 各类均衡但天花板~48%

**48%可能接近CNN在板端SDK热力图数据上的天花板**，原因：
1. 热力图类内cos_sim=0.97, 类间=0.965, 差异仅0.005
2. CNN的归纳偏置(局部性+平移不变性)对平滑高斯blob几乎无用
3. 传统ML(RF=47.7%, SVM=54.5%)在PCA空间反而更适合

### 3.2 中性偏执 — 集成场景下的致命问题 (V19g新发现)

V19f虽然ValAcc=47.9%，但在板端Expert集成后整体暴跌16pp(64.6%→48.3%)：
- CNN把过半厌恶(53%)和悲伤(57%)判成中性
- Expert的强类路由依赖CNN正确预测，CNN误判"中性"时覆盖了SVM7的正确结果
- **V15(30.6%)反而没这个问题**，因为V15中性recall仅12%，几乎不预测中性

**教训**: CNN独立ValAcc高不代表集成后好。误判方向比整体准确率更重要。

### 3.3 类别平衡的四大陷阱

| 陷阱 | 表现 | 解法 |
|------|------|------|
| 双重平衡(FocalLoss alpha+Sampler) | 厌恶偏83% | 只用一种 |
| 极端权重(1/count) | 中性偏81% | 用根号权重1/sqrt |
| 无平衡 | 恐惧0%、厌恶0.6% | 必须有温和平衡 |
| Label Smoothing加剧中性偏执 | LS=0.1时中性过度自信 | 有中性偏执时降LS到0.05或0 |

### 3.3 悲伤类是终极难点

5次训练悲伤从未超过19.4%。原因：
- 悲伤热力图和中性极度相似(都是"无表情"→热力图分布接近)
- V15 CNN悲伤4%, 任务书数据也显示CNN悲伤始终<5%
- **这可能是CNN结构固有的盲区**，无论怎么调参都难突破
